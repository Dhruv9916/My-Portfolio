# My Portfolio

Welcome to my personal portfolio website! This website showcases my skills, projects, and experiences, giving you a look at what I can do. It's designed using only **HTML**, **CSS**, and **JavaScript**.

## Features

- **Introduction**: A brief overview of who I am and what I'm passionate about.
- **Skills**: A section that highlights my core technical skills.
- **Projects**: A showcase of the various projects I’ve worked on, with links to live demos and repositories.
- **Contact**: Easy-to-use contact section to get in touch with me.

## Technologies Used

- **HTML**: The structure of the website.
- **CSS**: Styling to make the website look clean and responsive.
- **JavaScript**: Interactive elements such as form validation, animations, and dynamic content.
